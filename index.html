Objetivo OVI

Con el desarrollo de este OVI se pretende ofrecer una herramienta de consulta sobre el diseño de sitios web, a través de las lecturas, actividades y videos recopilando información relevante la cual ayudara al estudiante a adquirir conocimientos básicos sobre los lenguajes más utilizados para el diseño de sitios web. Dentro de estas encontramos lenguajes como HTML y CSS3 siendo estos los pilares en la construcción de una página web.

Objetivos del curso

Conocer los lenguajes de etiquetas como HTML5 y CSS3, con el fin de implementar el uso de estos a través de la construcción de un OVI usando una temática seleccionada y orientada al área de la informática. Manejar la plataforma GIT para el desarrollo de aplicaciones orientadas al control de versiones y manejo de copias locales dentro del grupo de diseñadores que colaboran con la construcción y mejoramiento de las aplicaciones.

QUE ES UN SITIO WEB

Los sitios web es un conjunto de archivos, la gran mayoría creados con el estándar html, las cuales están relacionadas y comunes a un dominio en la www por sus siglas en inglés “World Wide Web” los sitios web se pueden clasificar en dos grupos estáticos y dinámicos, los primero son aquellos en los cuales podemos observar información sin que tenga alguna interacción con bases de datos, los sitios dinámicos tienen una interacción con bases de datos en los cuales recopilan, procesan y muestran información con los visitantes.

GIT

Git es un software de control de versiones diseñado por Linus Torvalds, pensando en la eficiencia y la confiabilidad del mantenimiento de versiones de aplicaciones cuando éstas tienen un gran número de archivos de código fuente. Hay algunos proyectos de mucha relevancia que ya usan Git, en particular, el grupo de programación del núcleo Linux. (recuperado de: https://es.wikipedia.org/wiki/Git)

GitHub

GitHub es una forja (plataforma de desarrollo colaborativo) para alojar proyectos utilizando el sistema de control de versiones Git. Utiliza el framework Ruby on Rails por GitHub, Inc. El código de los proyectos alojados en GitHub se almacena típicamente de forma pública, aunque utilizando una cuenta de pago, también permite hospedar repositorios privados. HTML5

HTML es un lenguaje de marcado que se utiliza para el desarrollo de páginas de Internet. Se trata de la sigla que corresponde a HyperText Markup Language, es decir, Lenguaje de Marcas de Hipertexto, que podría ser traducido como Lenguaje de Formato de Documentos para Hipertexto. Se trata de un formato abierto que surgió a partir de las etiquetas SGML (Standard Generalized Markup Language).
